# Version 6.8

## how to use
First, install dependencies:
### Windows
1. Download and install Python 3.10 from https://www.python.org/downloads/windows/
2. Run
```
start.bat 3
```

### Linux
1. Install Python 3.10 using your package manager, or run
```
sudo apt install python3-pip
```
2. Run
```
chmod +x start.sh
./start.sh 3
```

## Start CLI

### #Windows
1. Run
```
start.bat 1
```

### Linux
1. Run
```
./start.sh 1
```

## Start GUI

### Windows
1. Run
```
start.bat 2
```
### Linux
2. Run
```
./start.sh 2
```
